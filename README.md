# 作業二：文字雲 (Word Cloud Generator)

這是一個基於 Python Tkinter 開發的圖形化介面程式，支援貼上文字、讀取 `.txt` 檔、使用 **Hash / Dictionary** 統計詞頻，並將高頻詞繪製成文字雲。

## 專案資訊
* **作業名稱**：作業2 - 文字雲
* **開發語言**：Python 3.x
* **GUI 框架**：Tkinter
* **核心概念**：Hash Table（Python dictionary）

## 功能亮點
1. **圖形化介面**：使用者可直接在介面貼上文字，也可開啟 `.txt` 檔。
2. **使用 Hash 統計詞頻**：本程式使用 Python 的 `dictionary` 作為 **hash table**，記錄每個字詞出現次數。
3. **產生文字雲**：依照出現頻率挑出前 N 個高頻詞，並以不同大小顯示在畫面上。
4. **存成圖片**：可將文字雲另存成 `.png` 檔案。
5. **加分功能**：
   - 英文 stop words 過濾
   - 中文片語初步擷取
   - 文字雲圖片輸出

## 核心邏輯說明

### 1. 文字前處理
程式會先整理文字內容，統一換行格式，再分別處理英文與中文：
* 英文部分：擷取單字並排除常見 stop words，例如 `is`、`are`、`the`。
* 中文部分：抓出連續中文字串，進一步擷取 2~4 字片語，提升中文結果的可讀性。

### 2. Hash 詞頻統計
為了符合課程要求，本程式核心統計方式如下：
```python
freq[word] = freq.get(word, 0) + 1
```
* `word` 作為 **key**
* 出現次數作為 **value**
* 每讀到一次字詞，就到 hash table 中更新次數

### 3. 文字雲繪製
* 依照字詞出現次數由高到低排序
* 取出前 N 個高頻詞
* 依頻率決定字體大小
* 將文字分散排入圓形區域中，形成文字雲

## 安裝與執行

### 環境需求
* Python 3.10+
* Tkinter
* matplotlib

### 執行程式
在終端機進入專案目錄後執行：
```bash
python3 main.py
```

## 使用說明
1. 在文字輸入區貼上一段文字，或點擊 **Open .txt File** 匯入文字檔。
2. 點擊 **Generate Word Cloud**，程式會統計高頻詞並繪出文字雲。
3. 右側欄位會顯示前 N 個高頻詞與出現次數。
4. 點擊 **Save Word Cloud** 可將結果存成圖片。
